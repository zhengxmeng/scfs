git add    添加文件
git status 查看状态
git commit 提交
git diff   查看修改了哪些文件
git branch 查看所有分支
git checkout 切换分支
git push   提交
git pull   拉代码

新建一个分支
$ git branch dev01  

分支是否创建成功
$ git branch 

切换新分支
$ git checkout dev01

如果不放心，还可以 $ git branch确认下
然后将你的改动提交到新分支上
$ git add . 
$ git commit -m "测试信息"

git status检查是否成功

切换到主分支
$ git checkout master 

将新分支提交的改动合并到主分支上
$ git merge newbranch  

然后就可以push代码了
$ git push -u origin master

最后还可以删除这个分支
$ git branch -D newbranch
